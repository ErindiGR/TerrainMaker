# Terrain Maker:
Terrain Maker is a terrain generator that simulates erosion.

* std_image:
    * link: https://github.com/nothings/stb
    * license(MIT): https://github.com/nothings/stb/blob/master/LICENSE

# Build:

release:

    sh build.sh

debug:

    sh build_debug.sh
